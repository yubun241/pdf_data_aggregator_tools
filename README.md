# PDF Page Reorder Tool (PDFページ順整理ツール)

## 概要
このツールは、両面同時スキャンができないスキャナーを使用して、大量の書類をデジタル化する作業を効率化するために作成しました。

「表面のみを一気にスキャン（前半データ）」し、次に「裏面のみを一気にスキャン（後半データ）」した2つのPDFファイルを読み込み、正しいページ順（奇数・偶数ページ）に並べ替えて1つのファイルに集計します。

## 開発の背景
紙の書類をスキャナーで取り込む際、両面印刷の書類を一枚ずつ手作業で裏返すのは非常に手間がかかります。
本ツールを使用することで、表面と裏面を別々にまとめてスキャンしても、後から自動で正しい順番に結合することが可能になります。

## 仕様
- **前半データ (Front side):** 表面のみを連続スキャンしたデータ。集計後は**奇数ページ**（1, 3, 5...）に配置されます。
- **後半データ (Back side):** 裏面のみを連続スキャンしたデータ。集計後は**偶数ページ**（2, 4, 6...）に配置されます。

## 使い方
1. 表面（前半）のPDFと裏面（後半）のPDFを用意します。
2. 本プログラムを実行します。
3. 自動的にページが交互に組み合わさり、正しい順序のPDFが出力されます。

## 動作環境
- Python Python 3.9.8

## フォルダ構造
リポジトリ内を以下のように構成します。
my-pdf-tool/ (リポジトリ名)
├── src/            # プログラム本体 (main.py)
├── data/           # 入力するPDF（表面.pdf, 裏面.pdf）を入れる
├── temp/           # 処理中の一次保存場所
├── result/         # 完成したPDFが出力される場所
├── .gitignore      # GitHubに上げないファイルを指定（tempやresultを除外）
└── README.md       # ツールの説明書
